# Scoop CN Mirror Bucket

国内镜像加速的 Scoop Bucket，将软件下载地址替换为 NJU、TUNA 等国内镜像站。

## 添加此 Bucket

```powershell
scoop bucket add cn-mirror https://gitee.com/axinger-scoop/cn-mirror.git
```

## 安装软件

```powershell
scoop install cn-mirror/temurin21-jdk
```
